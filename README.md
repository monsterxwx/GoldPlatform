# 🛠️ 创作盒子 | 自媒体与职场创作者的「超级排版工厂」
*(Xiaohongshu Cover, WeChat Article & Markdown Resume Generator)*

一个极简、优雅、纯前端构建的多功能自媒体与职场图文生成工具箱。基于 **Vue 3** + **Vite** + **UnoCSS** 打造，集成**小红书无脑封面图生成**、**微信公众号长文沉浸式排版**以及**极简主义 Markdown 简历排版**三大核心引擎。

支持一键打通 **DeepSeek AI 模型**，写字只管写，排版优化、格式美化甚至内容润色全部交给 AI 自动接管！无需注册，数据不经过任何第三方服务器（AI 服务除外），配置与文本本地缓存记忆，绝对隐私、安全且轻量。

---

## 🌟 核心引擎 (Core Engines)

### 📱 一、小红书图文封面生成器 (Cover Engine)
*为爆款而生的小红书无痛配图工具*
- **双模设计自由切换**：
  - **文字排版模式**：左侧纯文本 Markdown 输入，右侧毫秒级渲染出符合小红书视觉风格的爆款图文封面。
  - **图片加字模式**：支持一键上传 3:4 背景图，自由添加浮空标题并支持鼠标任意拖拽定位。
- **大师级视觉预设**：右侧默认渲染带“高级弱阴影”与“思源字体族”的图文封面，零设计基础也能输出极具高级感的视觉排版。
- **3 倍超高清无损导出**：通过底层按原生 3 倍像素比 (`pixelRatio: 3`) 进行无损渲染导出 `1080x1440` 小红书完美比例封面图片，确保在任何视网膜屏幕（Retina）设备上显示都极致清晰。

### ✍️ 二、公众号高定长文排版引擎 (Article Engine)
*让 Markdown 文章完美融入微信生态，回归纯粹的思想书写*
- **真正的所见即所得**：左侧 Markdown 实时编辑，右侧手机壳模型毫秒级同步渲染真实观感效果。
- **5 套顶流自媒体美学主题零延迟切换**：
  - 💎 **科技职场风格**：专业严谨，带有强视觉引导的高亮卡片，适合行业干货。
  - 🍏 **苹果极简风格 (Minimal)**：克制的高级感，极具呼吸感的字号间距与留白设计。
  - 💌 **情感走心风格 (Emotion)**：暖调樱花粉、温和的圆角与引用框，带给读者倾诉的安全感。
  - 📘 **深度技术风格 (Indigo)**：深度长文标配，优雅的暗蓝色系与完整的技术大表单支持。
  - 💻 **极客开发风格 (Geek)**：带有 Mac 终端红黄绿顶栏风格的代码域、专属的行内代码胶囊保护。
- **纯前端安全复制协议**：渲染完毕后，点击【🔗 一键复制排版】，系统将采用原生流媒体协议将 `text/html` 的带样式富文本写入剪贴板，您只需在公众号后台直接 `Ctrl+V`，没有任何数据通过后台 API 传输，绝对隐私。

### 📝 三、极简主义 Markdown 简历排版引擎 (Resume Engine) `[NEW]`
*用写代码的纯粹方式，排版出极致优雅的个人简历*
- **所见即所得的极速预览**：左侧编写 Markdown 简历内容，右侧标准 A4/简历版面毫秒级实时刷新渲染，简历变化一目了然。
- **极致的 PDF 导出与打印优化**：
  - 采用创新的 **DOM 深度克隆与脱离文档流打印技术**，一键点击【导出为 PDF】直达系统打印。
  - 针对纸张空间进行极致微调（字号、行高、上下边距），支持智能避免内容跨页截断 (`page-break-inside: avoid`)，输出完美的一页纸/多页纸高解析度 PDF 简历。
- **现代排版美学与细节把控**：默认搭载精选的 Helvetica Neue / PingFang SC 无衬线现代字体，采用干练的侧边指示彩条 (`h2::before`)，优雅的链接下划线悬浮反馈，以及顶部个人基础信息自动网格/间距对齐，确保简历质感拉满。
- **100% 本地化安全隐私保障**：简历数据完全存储于本地浏览器缓存，无需注册登录，绝无简历数据上传服务端的风险，彻底打消个人敏感隐私泄露顾虑。

### 🤖 四、DeepSeek 引擎级整合 (AI Core)
*双向赋能，让写作与排版如虎添翼*
无需复杂配置，点开编辑器内的密钥输入框，绑定个人的 DeepSeek API Key 即可唤醒智能辅助大军：
- **📐 一键排版**：你的内容全部连在一起了？点击 AI，瞬间帮你把文章加上优雅的 Markdown 骨架结构，不改变你的一字一句，只做物理结构调整。
- **✨ AI 优化/润色**：输入简陋的大纲或者草稿，指令直达云端，返回一篇结构完美、措辞优美的高配自媒体长文/小红书爆款文案。

## 🚀 快速启动 (Getting Started)

本项目使用 [Vue 3](https://vuejs.org/) + [Vite](https://vitejs.dev/) + [pnpm](https://pnpm.io/) 开发。

1. **安装项目依赖**
   ```bash
   # 推荐使用 pnpm
   pnpm install
   ```

2. **启动开发服务器**
   ```bash
   pnpm dev
   ```

3. **立刻使用**
   默认访问 `http://localhost:5173`。所有功能即点即用，AI 配置、简历内容和封面拖拽位置全持久化在您的本地 LocalStorage 中。

## 💻 核心技术栈 (Tech Stack)

- **[Vue 3](https://v3.vuejs.org/)** - 渐进式前端响应式框架，极致性能与组合式 API 体验。
- **[Vite 6](https://vite.dev/)** - 新一代超快速的前端构建工具。
- **[UnoCSS](https://unocss.dev/)** - 高性能原子化 CSS 引擎，极速、按需加载，完美适配各种复杂布局。
- **[marked.js](https://github.com/markedjs/marked)** - 构建强大自定义 Token 劫持解析能力的长文排版与简历渲染引擎核心。
- **[html-to-image](https://github.com/bubkoo/html-to-image)** - 用来截取 DOM 元素并原画质输出下载小红书保存图像的救星。
- **[Pinia](https://pinia.vuejs.org/)** - 现代 Vue 状态管理，并使用持久化插件实现本地配置和偏好的无缝保存。
- **[TypeScript](https://www.typescriptlang.org/)** - 强类型约束，提升代码的健壮性与可维护性。

## 🔒 隐私与安全声明 (Privacy & Security)

本项目是一项 **100% 纯前端构建** 的开源工具：
- **无后端服务器**：项目不含任何收集个人信息的后端代码，不设数据库，不要求用户注册或登录。
- **数据本地化**：您的所有排版文本、简历内容、上传的背景图及 DeepSeek API Key 均保存在您个人浏览器的 `LocalStorage` 中。
- **AI 传输通道**：若启用 DeepSeek 智能辅助，您的 API 密钥与待润色内容将通过加密 HTTPS 协议直连官方 DeepSeek 端点，不经过任何中转服务器。
- **极速粘贴保障**：生成的公众号富文本采用原生剪贴板机制写入，未经过任何远程服务器。
